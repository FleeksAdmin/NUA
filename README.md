# NUA

Инструкция по запуску `main.py` на Windows.

## О проекте

Проект написан **FleeksAdmin**.

Скрипт работает через Selenium:
- либо подсвечивает правильные ответы;
- либо делает автопрохождение (если выбран режим auto).

Ранее софт был приватным, но FleeksAdmin решил выложить его публично.

В проекте есть локальный OCR:
- числовой OCR;
- OCR для текста/языков.

OCR используется для распознавания текста или чисел на изображениях в вопросах/вариантах ответа.

Используйте софт с умом. Автор не несет ответственность за последствия использования.

## Быстрый старт

1. Запустите `setup.bat` **от имени администратора**.
2. Дождитесь завершения. В конце будет сообщение:
   `DONE. READE README.MD TO CONTIUNE`
3. Откройте сайт с тестом и дойдите до страницы, где уже вопросы.
4. Нажмите `F12` (DevTools).
5. Перейдите во вкладку `Network`.
6. Обновите страницу (`F5`) с открытым `Network`.
7. Нажмите на значок скачивания HAR в панели `Network`.
8. Сохраните файл в эту же папку со скриптом под именем:
   `naurok.com.ua.har`
   (**Это самый важный шаг! Без HAR-файла скрипт не увидит ответы в коде страницы.**)
9. Запустите `fast.bat`.
10. Вставьте ссылку на тест.
11. Выберите режим:
   - `1` = автопрохождение (`--auto`)
   - `2` = только подсветка ответов
12. Выберите провайдер:
   - `1` = g4f (`--gpt`, Gemini 2.5 Flash Lite) - лучший бесплатный вариант
   - `2` = DeepSeek (`--deep`) - платный API и обычно менее эффективный
   - `3` = g4f (`--gpt`, Groq + GPT OSS 120B) - альтернативный бесплатный вариант
   - `4` = g4f (`--gpt`, NVIDIA + GPT OSS 120B) - альтернативный бесплатный вариант
13. Если выбрали DeepSeek, введите ваш API-ключ в `fast.bat`.

## Что делает fast.bat

`fast.bat` только запускает `main.py` с флагами.  
Фактически используется команда:

```bash
python main.py --har "naurok.com.ua.har" --url "<ваша_ссылка>" [--gpt|--deep] [--auto]
```

## Ручной запуск (по желанию)

Обычный запуск:

```bash
py -3.11 main.py --url "https://example.com/test" --gpt
```

Автопрохождение:

```bash
py -3.11 main.py --url "https://example.com/test" --gpt --auto
```

DeepSeek (нужно указать API-ключ):

```bash
py -3.11 main.py --url "https://example.com/test" --deep --deep-key "sk-your-key"
```

## Флаги main.py

- `--har` путь к HAR (по умолчанию: `naurok.com.ua.har`)
- `--url` ссылка на тест (обязательно)
- `--gpt` использовать g4f
- `--deep` использовать DeepSeek
- `--g4f-endpoint` endpoint для g4f: `gemini`, `groq` или `nvidia` (по умолчанию: `gemini`)
- `--gpt-model` модель для выбранного g4f endpoint (по умолчанию: автоматически)
- `--auto` автоматически кликать ответы
- `--deep-key` API-ключ DeepSeek (если не указан, используется переменная `DEEPSEEK_API_KEY`)
- `--ocr-langs` OCR-языки через запятую (по умолчанию: `eng`)

## Важно

- Файл `naurok.com.ua.har` должен лежать рядом с `main.py`.
- Нужен стабильный интернет.
- Для Selenium должен быть установлен Google Chrome.
